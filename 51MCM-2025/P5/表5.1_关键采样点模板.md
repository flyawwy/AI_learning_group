## 使用说明

1. 运行相应的命令获取完整的关键采样点分析结果：
   ```
   python P2.py --p5
   python P3.py --p5
   ```

2. 查看`./P5/P2/`或`./P5/P3/`目录下的`关键采样点分析报告.md`获取详细结果。

3. 关键采样点的选择基于以下原则：
   - 流量变化显著的点（导数变化大）
   - 流量峰值和谷值
   - 模型转折点
   - 信号灯变化点（对于P3问题）
